Configurer un Node USB
Art-Net / DMX

Vous avez un Node 1 Univers Art-Net/sACN (XLR 3 points) branché en USB ? Voici la marche à suivre complète pour le régler et le faire dialoguer avec MyStrow — de la vérification du boîtier jusqu'au premier fader.

Par MyStrow · Publié le 13 juillet 2026 · ⏱ Lecture : 6 min
Commencer la config → Télécharger MyStrow ↓
Avant de commencer

1. Vérifier que c'est bien un Node Art-Net

Première chose à faire : regardez l'inscription juste au-dessus de la prise USB du boîtier.

✅ Si c'est écrit « USB NODE DMX » → vous êtes au bon endroit, c'est bien un node Art-Net, suivez ce guide.

⚠️ Si c'est écrit « OPTO OPEN DMX » → il ne s'agit PAS d'un node Art-Net, mais d'une interface USB / DMX classique. La configuration est différente : référez-vous au guide d'installation USB/DMX classique.

Une fois confirmé qu'il s'agit bien d'un USB NODE DMX, branchez le boîtier et assurez-vous qu'il est bien alimenté en USB (la LED du boîtier doit être active).

Configuration du boîtier

2. Régler le node dans DMX TOOLS

La configuration du node se fait via l'outil web d'ElectroConcept. Rendez-vous sur :

DMX TOOLS — ElectroConcept

L'utilitaire de configuration en ligne pour votre node. Votre boîtier doit être branché et alimenté en USB.

Ouvrir DMX-tools.electroconcept.fr ↗

Appliquez les réglages suivants :

A

Sortie DMX

Renseignez exactement ces valeurs (identiques à la capture ci-dessous) : DMX Mode : OUT · Universe OUT : 0 · Universe IN : 1 · OUT Mode : Fixed · FPS Fixed : 40.
(DMX Mode OUT = c'est le PC qui envoie vers vos projecteurs ; Universe OUT 0 = l'univers Art-Net émis par MyStrow.)

B

Réseau du boîtier

Node IP : 2.0.0.15 (à changer — le boîtier sort d'usine sur 2.0.0.1). DHCP IP : 2.0.0.2 (réglage du boîtier ; côté PC on utilisera plutôt une IP fixe, voir étape 3). Netmask : 255.0.0.0.

C

Appliquer

Validez, puis débranchez et rebranchez le câble USB pour que le boîtier reparte sur sa nouvelle adresse (2.0.0.15).

Côté ordinateur

3. Régler le PC en IP fixe

On préfère toujours une IP fixe sur le PC — c'est plus stable et plus fiable qu'une attribution automatique. Ouvrez les propriétés de votre carte réseau → Protocole Internet version 4 (TCP/IPv4) → Propriétés, cochez « Utiliser l'adresse IP suivante » et renseignez :

📍 Adresse IP : 2.0.0.1 🎭 Masque : 255.0.0.0 🚫 Passerelle : vide

⚠️ Le piège à éviter : le boîtier sort d'usine sur 2.0.0.1. Comme on l'a déplacé sur 2.0.0.15 à l'étape 2, le PC peut rester sur 2.0.0.1 — les deux adresses sont différentes, aucun conflit. La règle : PC et node ne doivent jamais avoir la même adresse. À ne jamais utiliser non plus : 2.0.0.0 (réseau) et 2.255.255.255 (broadcast).

Dans le logiciel

4. Paramétrer la sortie dans MyStrow

Ouvrez MyStrow et allez dans Connexion → Sortie DMX. La fenêtre « Paramétrer la sortie DMX » s'ouvre : choisissez l'onglet Sortie Node (boîtier réseau Art-Net — ElectroConcept, MA Lighting, etc.).

Si le boîtier n'est pas encore détecté, cliquez sur ⚙ Configurer la connexion réseau (MyStrow peut régler la carte réseau à votre place — c'est l'équivalent automatique de l'étape 3). Une fois la carte réseau OK, cliquez sur Connecter. MyStrow cible 2.0.0.15 par défaut, vous n'avez rien d'autre à saisir.

Validation

5. Vérifier que tout fonctionne

1

Ping du boîtier

Invite de commande → ping 2.0.0.15. Le boîtier doit répondre (0 % de perte).

2

Adresse du PC

ipconfig → la carte réseau du node doit afficher une IPv4 en 2.0.0.x (typiquement 2.0.0.2), masque 255.0.0.0.

3

Test final

Dans MyStrow, l'indicateur DMX doit passer au vert. Montez un fader : vos projecteurs doivent s'allumer. 🎉

Dépannage. « Connexion perdue » juste après avoir changé le Node IP ? C'est normal, le boîtier a changé d'adresse → débranchez/rebranchez l'USB, fermez et rouvrez DMX TOOLS. Le node n'est plus détecté ? Coupez l'alimentation (USB débranché 10 s), rebranchez, puis ping 2.0.0.15 et ping 2.0.0.1 pour voir où il répond. PC en 169.254.x.x ou DHCP à 2.0.0.0 ? Le DHCP n'a pas donné d'adresse valide → remettez 2.0.0.2 et faites ipconfig /release puis ipconfig /renew.

💡 En résumé : Node = 2.0.0.15, PC = 2.0.0.1, Masque = 255.0.0.0, en DMX OUT / Univers 0. Réglé une seule fois, votre node USB est prêt : à chaque spectacle, il suffit de le brancher et de cliquer « Connecter ».
