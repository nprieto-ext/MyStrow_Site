5 000+ profils prêts à l'emploi

Un fixture DMX (ou profil de projecteur) décrit exactement comment votre appareil fonctionne : combien de canaux il utilise, ce que chaque canal contrôle (rouge, vert, bleu, strobe, pan, tilt…), et dans quel ordre. Sans ce profil, le logiciel ne peut pas envoyer les bonnes valeurs à votre matériel.

MyStrow embarque directement plus de 5 000 profils de projecteurs couvrant les grandes marques du marché — Chauvet, ADJ, Eurolite, Stairville, Robe, Martin, Showtec, Varytec, et des centaines d'autres. La grande majorité des projecteurs courants sont déjà là, sans rien à installer.

Installation requise

Pour utiliser un fixture intégré, rendez-vous dans le Panneau Admin → Bibliothèque de fixtures, tapez le nom ou la marque de votre appareil dans la barre de recherche, et ajoutez-le à votre patch en un clic.

Si votre modèle exact n'est pas trouvé, ne paniquez pas — la suite de cet article vous montre comment résoudre ça rapidement.

Solution 1 — La plus simple

Télécharger un profil XML sur FixtureShare

La ressource la plus complète et la plus fiable pour trouver un fixture DMX gratuit, c'est FixtureShare, la base de données officielle de MA Lighting — l'un des fabricants de consoles les plus respectés du secteur.

FixtureShare MA Lighting

Des milliers de fichiers XML de fixtures vérifiés, organisés par marque et modèle. Téléchargement gratuit, sans compte.

Ouvrir FixtureShare ↗
1

Rechercher votre projecteur

Sur fixtureshare.malighting.com, tapez la marque ou le modèle de votre projecteur dans la barre de recherche. Vous obtiendrez une liste de fichiers correspondants avec leur version et leur date de mise à jour.

2

Télécharger le fichier XML

Cliquez sur le fixture qui correspond le mieux à votre appareil (vérifiez le nombre de canaux et le mode DMX dans la description). Téléchargez le fichier — il porte l'extension .xml. MyStrow supporte le format XML GrandMA.

3

Importer dans MyStrow

Dans MyStrow, allez dans Panneau Admin → Bibliothèque de fixtures → Importer (ou utilisez le menu Fichier → Importer fixture). Sélectionnez votre fichier XML téléchargé. Le fixture apparaît immédiatement dans votre bibliothèque, prêt à être utilisé dans votre patch.

Conseil : Si votre projecteur a plusieurs modes (3 canaux, 5 canaux, 8 canaux…), cherchez le fichier qui contient tous les modes. Dans MyStrow, vous choisirez le mode actif directement dans les paramètres du projecteur lors du patch.

💡 Fichier introuvable ? Si votre fixture n'est pas dans la bibliothèque intégrée, cherchez-le au format .xml sur FixtureShare MA Lighting — il est presque toujours disponible.

Solution 2 — Pour les cas spéciaux

Créer son propre fixture dans MyStrow

Vous avez un projecteur exotique, un appareil sur mesure, ou une machine dont le manuel ne corresponds à aucun profil existant ? MyStrow inclut un éditeur de fixtures qui vous permet de définir vos propres profils canal par canal, sans quitter l'application.

1

Ouvrir l'éditeur de fixtures

Dans MyStrow, allez dans Panneau Admin → Bibliothèque de fixtures → Nouveau fixture (ou cliquez sur le bouton « + »). L'éditeur s'ouvre avec un profil vierge.

2

Renseigner les informations de base

Saisissez le nom du fabricant, le nom du modèle, et sélectionnez le type d'appareil (PAR LED, lyre, barre LED, stroboscope…). Ces informations serviront à retrouver le fixture dans la bibliothèque.

3

Définir les canaux DMX

Ouvrez le manuel de votre projecteur à la page du tableau des canaux DMX. Pour chaque canal, indiquez à quoi il correspond : Dimmer, Rouge, Vert, Bleu, Blanc, Ambre, Strobe, Pan, Tilt… MyStrow dispose d'une liste prédéfinie de types de canaux couvrant tous les usages courants.

4

Sauvegarder et utiliser

Cliquez sur Enregistrer. Votre fixture personnalisé est instantanément disponible dans la bibliothèque, au même titre que les 5 000 fixtures intégrés. Vous pouvez aussi l'exporter en fichier .xml pour le partager ou le sauvegarder.

Astuce : Consultez le manuel de votre projecteur à la section "DMX Channel Configuration" ou "Mode d'emploi DMX". Tous les projecteurs DMX disposent d'un tableau indiquant le canal, sa fonction et les plages de valeurs (0-255). C'est tout ce dont vous avez besoin pour créer le profil.
