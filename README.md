# MyStrow — Site Web

Site vitrine de **[MyStrow](https://mystrow.fr)**, logiciel de contrôle lumière DMX professionnel pour Windows et macOS.

## Pages

| Page | Description |
|---|---|
| `index.html` | Page d'accueil — présentation, fonctionnalités, tarifs, FAQ |
| `logiciel-lumiere-dmx-ia.html` | Logiciel Lumière DMX — patch, groupes, compatibilité projecteurs |
| `ia-lumiere.html` | IA Lumière — éclairage réactif au son, analyse BPM/fréquences |
| `controleur-akai-apc-mini.html` | Contrôleur AKAI APC mini mk2 — pads, faders, plan de feu |
| `programmation-lumiere.html` | Programmation lumière — timeline, keyframes, clips |
| `player-audio-video.html` | Player audio & vidéo intégré |
| `plan-de-scene.html` | Plan de scène interactif |
| `telecharger.html` | Téléchargement — Windows & macOS |
| `contact.html` | Contact & formulaire bugs/suggestions |
| `confidentialite.html` | Politique de confidentialité |

## Stack

- HTML/CSS/JS vanilla — aucune dépendance, aucun framework
- Fonts : Bebas Neue, Barlow Condensed, Inter (Google Fonts)
- Formulaire de contact : [Formspree](https://formspree.io)
- Vidéos : WebM VP9
